```
下载
npm i hyh-cli -g

目前支持的框架有 vue|react|uniapp

创建项目
hyh create <projectName>

创建一个页面
hyh addpage <frame> <pageName>

创建一个组件
hyh addcpn <frame> <cpnName>

创建一个node模块
hyh addmodule <moduleName>

自定义路径
hyh addcpn <frame> <cpnName> -d <path>

切换npm源
hyh origin

查看配置文件: 可以在配置文件中更改仓库地址、默认路径
hyh config

帮助
hyh --help
```
